## Guide 

*Run App Dev*

```sh
bun run dev
```

*Production mode*

> after build

```sh
bun run start
```

*Run custom port*

```sh
bun run dev -p 3003
```

*Run on app on local network*

```sh
bun run start -H 0.0.0.0 -p 3003
```
